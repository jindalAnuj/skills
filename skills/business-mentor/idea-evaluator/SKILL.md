# Idea Evaluator

Combat shiny object syndrome by scoring ideas against proven business criteria before committing time or money.

## When to Use

- You have a new business idea and want to evaluate it
- You're comparing multiple opportunities
- You feel the pull of a "shiny object" and need grounding
- You want to validate whether to continue an existing project

## Evaluation Framework

Score each criterion 1-5, then calculate weighted total.

### Scoring Criteria

| Criterion | Weight | Question to Answer |
|-----------|--------|-------------------|
| **Efficiency Multiplier** | 15% | Does this reduce customer costs measurably? (1=no savings, 5=50%+ savings) |
| **Success Alignment** | 20% | Does revenue grow as customer succeeds? (1=flat fee, 5=pure revenue share) |
| **Tech Reusability** | 15% | Can this tech power other products? (1=single use, 5=platform for 10+ products) |
| **Viral Mechanics** | 15% | Does product promote itself? (1=no virality, 5=every use = free marketing) |
| **GTM Clarity** | 15% | Is distribution channel clear? (1=no idea, 5=proven marketplace/SEO) |
| **Monetization Clarity** | 10% | Is path to payment obvious? (1=hope for ads, 5=clear willingness to pay) |
| **Effort to Launch** | 10% | How fast to MVP? (1=6+ months, 5=weekend project) |

### Quick Score Calculator

```
Total = (Efficiency × 0.15) + (Success × 0.20) + (Tech × 0.15) + 
        (Viral × 0.15) + (GTM × 0.15) + (Monetize × 0.10) + (Effort × 0.10)
```

**Interpretation:**
- **4.0+**: Strong opportunity, proceed with conviction
- **3.0-3.9**: Viable but has gaps to address
- **2.0-2.9**: Significant risks, consider pivoting the concept
- **<2.0**: Pass, or fundamentally rethink

```markdown
| Success Alignment | _ | |
| Tech Reusability | _ | |
| Viral Mechanics | _ | |
| GTM Clarity | _ | |
| Monetization Clarity | _ | |
| Effort to Launch | _ | |

**Weighted Total:** ___

### Red Flags

- [ ] No clear customer pain point
- [ ] Requires massive user base to monetize
- [ ] Can't articulate who pays and why
- [ ] Tech is one-time use, no leverage
- [ ] No obvious distribution channel
- [ ] Competing with well-funded incumbents on features

### Green Flags

- [ ] Solves problem I personally have
- [ ] Clear willingness to pay (people already paying for alternatives)
- [ ] Built-in distribution (marketplace, virality, SEO)
- [ ] Tech compounds across products
- [ ] Can launch MVP in <30 days

### Decision

- [ ] **PROCEED** - Score ≥3.5 and no critical red flags
- [ ] **ITERATE** - Good core but needs refinement
- [ ] **PASS** - Move on, don't look back
```

## Shiny Object Syndrome Check

Before evaluating a NEW idea, ask:

1. **What am I currently working on?** [Answer]
2. **What stage is it at?** [Idea/Building/Launched/Growing]
3. **Why am I considering switching?** [Answer honestly]
4. **Is this new idea actually better, or just newer?**

**Rule:** Unless new idea scores 1.5+ points higher than current project, STAY THE COURSE.

## Quick Rejection Filters

Reject immediately if:

- [ ] Revenue depends entirely on viral/social growth (unless you have proven viral expertise)
- [ ] Requires building audience before monetizing (unless audience-building IS the product)
- [ ] No path to $10K MRR without 100K+ users
- [ ] Core value proposition is "like X but cheaper"
