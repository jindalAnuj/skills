# Visual AI Prompting — Glossary

| Term | Definition |
|---|---|
| **Anamorphic** | Wide cinema lens format (2.39:1) creating oval bokeh and horizontal lens flares |
| **Aspect Ratio** | Width:Height ratio of the frame (e.g. 16:9, 2.39:1) |
| **Bird's Eye View** | Camera positioned directly overhead |
| **Bleach Bypass** | Color grade that retains silver, creating desaturated high-contrast look |
| **Bokeh** | Out-of-focus blur in background; wide aperture + long focal length |
| **Butterfly Lighting** | Overhead front light creating a butterfly shadow below the nose |
| **Caustics** | Patterns of refracted/reflected light through transparent surfaces |
| **CFG Scale** | Stable Diffusion parameter controlling how strictly the model follows the prompt |
| **Chiaroscuro** | High-contrast light and shadow technique (Italian: "light-dark") |
| **Chromatic Aberration** | Color fringing on edges; lo-fi / vintage lens effect |
| **Cinematographer / DP** | Director of Photography — person who designs and executes the visual look |
| **Color Grade** | Overall color treatment applied to an image or clip |
| **Color Temperature** | Warmth/coolness of light measured in Kelvin |
| **Crane Shot** | Camera mounted on a crane arm; moves vertically or arcs |
| **Depth of Field (DOF)** | Range of distance in sharp focus; shallow DOF = blurry background |
| **Director Reference** | Naming a film director to invoke their visual style |
| **Dolly** | Camera physically moves toward or away from the subject on a track |
| **Dutch Tilt** | Camera tilted diagonally to create tension or unease |
| **EWS** | Extreme Wide Shot — subject tiny against vast environment |
| **Film Grain** | Organic texture from analog film stock |
| **Film Stock** | Specific analog film (e.g. Kodak Vision3, Fujifilm Velvia) with distinct color characteristics |
| **Focal Length** | Lens length in mm; affects field of view and perspective compression |
| **Foreground Layering** | Placing elements at different depths for dimensionality |
| **God Rays** | Visible shafts of light through atmosphere (fog, dust, smoke) |
| **Golden Hour** | Warm directional light in ~60 minutes after sunrise or before sunset |
| **Handheld** | Camera held without stabilizer; creates organic movement |
| **High Angle** | Camera above subject, looking down |
| **High Key** | Bright, low-contrast lighting; white or light backgrounds |
| **Key Light** | Primary / main light source in a scene |
| **Leading Lines** | Compositional element guiding the eye toward the subject |
| **Lens Flare** | Optical artifact when light hits the lens directly |
| **LoRA** | Low-Rank Adaptation — fine-tuned model layer for specific styles in SD/Flux |
| **Low Angle** | Camera below subject, looking up — conveys power |
| **Low Key** | Dark, shadow-dominant lighting |
| **Macro** | Extreme close-up lens for tiny subjects |
| **Negative Prompt** | In Stable Diffusion: terms to exclude from the output |
| **Negative Space** | Empty area around a subject; creates emotional weight |
| **OTS** | Over-the-Shoulder Shot — camera behind one character looking toward another |
| **Pan** | Camera rotates horizontally left or right |
| **Rack Focus** | Shifting focus between two subjects in the same shot |
| **Rembrandt Lighting** | Portrait lighting with a small triangle of light on the shadowed cheek |
| **Rim Lighting** | Backlight creating a glowing edge on the subject |
| **Rule of Thirds** | Compositional grid dividing frame into 9 equal parts; subject on intersection points |
| **Silhouette** | Subject completely backlit; no detail visible |
| **Shallow DOF** | Narrow focus plane; background is blurred |
| **Shot Type** | Framing of the subject relative to the frame (CU, MS, WS, etc.) |
| **Silhouette** | Subject backlit with no fill, visible only as a dark shape |
| **Split Lighting** | Exactly half the face illuminated, half in shadow |
| **Steadicam** | Camera stabilizer enabling smooth moving shots |
| **Style Reference** | Image or keyword used to guide the visual aesthetic |
| **Tilt** | Camera rotates vertically up or down |
| **Tracking Shot** | Camera moves alongside a moving subject |
| **Volumetric Lighting** | Visible light rays through particles (fog, dust, smoke, rain) |
| **Whip Pan** | Ultra-fast horizontal camera rotation; creates motion blur transition |
| **Wide Angle** | Short focal length (14–35mm) capturing a broad field of view |
| **Worm's Eye View** | Extreme low angle looking up — overwhelming, imposing |
