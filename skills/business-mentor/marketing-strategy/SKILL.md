# Marketing Strategy

Build a marketing plan that matches your resources, skills, and product type.

## When to Use

- You have a product but no marketing plan
- You're choosing between marketing channels
- You're planning a product launch
- Current marketing isn't working
- You need to market with limited budget/time

## Go-To-Market Playbooks

### Playbook A: Marketplace Launch (QuickBooks, Shopify, etc.)

```markdown
## Marketplace GTM Checklist

### Pre-Launch (Week -2 to -1)
- [ ] Optimize listing title with keywords
- [ ] Write compelling description (problem → solution → benefits)
- [ ] Create screenshots showing key features
- [ ] Prepare demo video (60-90 seconds)
- [ ] Set up onboarding email sequence

### Launch Week
- [ ] Submit to marketplace
- [ ] Ask 5-10 beta users for honest reviews
- [ ] Post on relevant subreddits (not spammy)
- [ ] Share in niche Facebook groups
- [ ] Announce on Twitter/LinkedIn

### Growth Phase (Month 1-3)
- [ ] Respond to every review (positive and negative)
- [ ] Create comparison content (vs competitors)
- [ ] Build SEO content for "[problem] [marketplace] app"
- [ ] Cross-promote with complementary apps
- [ ] Iterate on listing based on search analytics
```

### Playbook B: Content/SEO Launch

```markdown
## Content Marketing GTM Checklist

### Foundation (Week 1-4)
- [ ] Keyword research: find 20-50 target keywords
- [ ] Create pillar content (3-5 comprehensive guides)
- [ ] Set up blog/content hub
- [ ] Implement basic on-page SEO

### Building (Month 2-3)
- [ ] Publish 2-4 articles per week
- [ ] Create comparison/alternatives posts
- [ ] Build internal linking structure
- [ ] Start guest posting for backlinks

### Acceleration (Month 4-6)
- [ ] Repurpose content for social
- [ ] Create lead magnets from top content
- [ ] Build email list from traffic
- [ ] Launch product to email list
```

### Playbook C: Viral/Product-Led Launch

```markdown
## Viral GTM Checklist

### Viral Mechanics to Implement
- [ ] Branding on output ("Made with [Product]")
- [ ] Share buttons with pre-filled text
- [ ] Referral rewards (both sides)
- [ ] Public profiles/portfolios
- [ ] Embeddable widgets

### Launch Strategy
- [ ] Soft launch to small group, iterate on virality
- [ ] Measure viral coefficient (invites per user)
- [ ] Optimize viral loops before scaling
- [ ] Product Hunt launch
- [ ] Hacker News Show HN
- [ ] Relevant subreddit launches

### Viral Coefficient Math
K = invites × conversion rate
- K > 1 = exponential growth
- K = 0.5-1 = sustainable with other channels
- K < 0.5 = need other channels primarily
```

### Playbook D: Cold Outreach (B2B)

```markdown
## Cold Outreach GTM Checklist

### Preparation
- [ ] Define ICP precisely
- [ ] Build list of 100 target companies
- [ ] Find decision-maker contacts (LinkedIn, Apollo, etc.)
- [ ] Write 3 email variations

### Outreach Sequence
Email 1: Problem-focused (Day 1)
Email 2: Social proof (Day 3)
Email 3: Value add (Day 7)
Email 4: Breakup email (Day 14)

### Metrics to Track
- Open rate target: >50%
- Reply rate target: >10%
- Meeting rate target: >3%

### Iteration
- [ ] Test subject lines (A/B)
- [ ] Test opening lines
- [ ] Test CTAs
- [ ] Track which personas respond best
```

## Launch Timing Strategy

### Soft Launch (Week 1-2)
- Release to small group (friends, early signups)
- Gather feedback, fix critical bugs
- Get initial reviews/testimonials

### Community Launch (Week 3-4)
- Reddit (find 3-5 relevant subreddits)
- Hacker News (Show HN for tools)
- Indie Hackers
- Niche communities

### Public Launch (Week 5-6)
- Product Hunt (Tuesday-Thursday best)
- Twitter/LinkedIn announcement
- Email list announcement
- Paid promotion (if budget allows)

## Marketing with Limited Resources

### Zero Budget Playbook

| Activity | Time Required | Expected Impact |
|----------|---------------|-----------------|
| Reddit participation | 30min/day | Medium-High |
| Twitter building | 30min/day | Medium |
| SEO content | 2-4hrs/week | High (long-term) |
| Cold DMs | 1hr/day | Medium |
| Community engagement | 1hr/day | High |

### Time-Constrained Playbook (2hrs/week)

Focus ONLY on:
1. One content piece per week (SEO)
2. Respond to all customer messages
3. One outreach/partnership attempt

### Money but No Time Playbook

| Delegation | Cost | Who |
|------------|------|-----|
| Content writing | $50-200/article | Upwork writers |
| Social management | $500-1K/mo | VA or agency |
| Paid ads | Variable | Self or agency |
| Link building | $100-500/mo | Outreach service |

## Common Marketing Mistakes

| Mistake | Fix |
|---------|-----|
| Trying all channels at once | Pick 2, do them well |
| Expecting instant results from SEO | Plan for 6-month horizon |
| No tracking/attribution | Set up analytics before marketing |
| Features over benefits | Lead with outcomes |
| Talking to everyone | Narrow your ICP |
| Giving up after one failed launch | Plan multiple launch moments |
